📸 How to Use

Step 1: Upload Context Files (Optional)

  • Drag & drop files (PDF, images, code, text)

Step 2: Describe Your Goal

Write a simple sentence describing what you want the AI to do:

“Analyze this research paper and generate a comprehensive literature review with key findings and methodology critique”

“Based on this code, generate complete unit tests with edge cases”

💡 Tips for Best Results

  1. Be specific in your goal description — “analyze the methodology” beats “read this paper”
  2. Upload relevant files — the tool uses file content to generate context-aware prompts
  3. Choose the right style — use “Structured” for complex tasks, “Simple” for quick ones
  4. Select target AI — each AI has different strengths; the tool optimizes accordingly
  5. Use Expert complexity for research or professional work
  6. Preview tab lets you see the rendered prompt before copying
