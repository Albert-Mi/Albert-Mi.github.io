下面的规则与题库完全统一，所有题目统一 **四个选项**：
**远光灯 / 近光灯 / 远近光灯交替 / 示廓灯+双闪**。  
（点击文末链接进入在线模拟考试。）

## 灯光规则

- **近光灯**：夜间直行通过路口；同方向近距离跟车；与机动车会车；在有路灯的道路上行驶；在照明良好的道路上行驶。  
- **远光灯**：夜间在照明不良条件下行驶；夜间在没有路灯条件下行驶。  
- **远近光灯交替**：夜间通过急弯、坡路、拱桥、人行横道；夜间超越前方车辆；夜间通过没有交通信号灯控制的路口。  
- **示廓灯+双闪**：路边临时停车；夜间在道路上发生故障。

## 题库（问法与考试一致）

> 选择正确的灯光操作（四选一）

1. 夜间直行通过路口 → **近光灯**  
2. 夜间同方向近距离跟车行驶 → **近光灯**  
3. 夜间与机动车会车 → **近光灯**  
4. 夜间在有路灯的道路上行驶 → **近光灯**  
5. 夜间在照明良好的道路上行驶 → **近光灯**  
6. 夜间在照明不良条件下行驶 → **远光灯**  
7. 夜间在没有路灯条件下行驶 → **远光灯**  
8. 夜间通过急弯 → **远近光灯交替**  
9. 夜间通过坡路 → **远近光灯交替**  
10. 夜间通过拱桥 → **远近光灯交替**  
11. 夜间通过人行横道 → **远近光灯交替**  
12. 夜间超越前方车辆 → **远近光灯交替**  
13. 夜间通过没有交通信号灯控制的路口 → **远近光灯交替**  
14. 路边临时停车 → **示廓灯+双闪**  
15. 夜间在道路上发生故障 → **示廓灯+双闪**
