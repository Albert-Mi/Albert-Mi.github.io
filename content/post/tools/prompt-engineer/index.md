## 🧪 Prompt Style Examples

### Structured (Role + Context + Task)

```
# Role
You are an expert data scientist specializing in...

# Context
You have been provided with a dataset containing...

# Task
1. Clean and validate the data
2. Perform exploratory analysis
3. Build a predictive model
...

# Output Format
Provide results in markdown with...
```

### Chain-of-Thought

```
Let's approach this step by step:

Step 1: First, examine the data structure...
Step 2: Then, identify potential issues...
Step 3: Next, apply appropriate transformations...
...
```

### System Prompt (for API)

```
You are a professional code reviewer. Your role is to...

When reviewing code, always:
- Check for security vulnerabilities
- Evaluate performance implications
- Suggest concrete improvements with code examples
...
```
